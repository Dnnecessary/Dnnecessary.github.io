# 需求文档

## 1. 应用概述

### 1.1 应用名称
墨迹文卡

### 1.2 应用描述
一款专注于内容排版的轻量级网页工具，帮助用户将纯文本快速转化为高颜值的长图卡片。提供极致的编辑体验与丰富的视觉模板，支持高度自定义的水印功能，满足用户多样化的分享需求。

## 2. 用户与使用场景

### 2.1 目标用户
- 需要快速生成精美文字卡片的内容创作者
- 希望将文本内容转化为可分享图片的用户
- 对排版美观度有要求的文档编辑者

### 2.2 核心使用场景
- 将Markdown文本转换为带样式的长图卡片
- 使用模板快速生成符合特定风格的内容卡片
- 添加个性化水印和页头页脚信息后导出分享

## 3. 页面结构与功能说明

### 3.1 页面结构

```
墨迹文卡主界面
├── 左侧导航栏
│   ├── 编辑
│   ├── 模板
│   ├── 自定义
│   ├── 小组件
│   ├── AI助手
│   ├── 帮助
│   └── 用户
├── 左侧内容面板（根据导航切换）
│   ├── 编辑面板
│   ├── 模板面板
│   ├── 自定义面板
│   └── 小组件面板
├── 中间编辑区
│   ├── 工具栏
│   └── Markdown编辑器
├── 右侧预览区
│   └── 卡片实时预览
└── 右上角导出按钮
    └── 导出弹窗
```

### 3.2 左侧导航栏

#### 3.2.1 导航项
- 以竖向排列的图标+文字形式展示
- 包含：编辑、模板、自定义、小组件、AI助手、帮助、用户
- 点击导航项切换左侧内容面板显示内容

### 3.3 编辑面板

#### 3.3.1 快捷工具栏
- 位于编辑器上方
- 提供可视化按钮：H（标题）、B（加粗）、斜体、无序列表、有序列表、引用、分割线、图片、表格、数学公式
- 点击按钮在编辑器光标位置插入对应Markdown语法

#### 3.3.2 Markdown编辑器
- 支持标准Markdown语法输入：标题、列表、引用、加粗、斜体等
- 实时解析并在右侧预览区渲染效果
- 支持LaTeX数学公式输入与渲染
- 支持代码块输入，自动识别编程语言并进行语法高亮

### 3.4 模板面板

#### 3.4.1 分类导航Tab
- 横向排列：推荐、全部、纯色、简约、创意、学术、商务、定制
- 每个Tab显示该分类下的模板数量，格式为「分类名 (数量)」
- 点击Tab切换显示对应分类的模板

#### 3.4.2 模板展示
- 以缩略图网格形式展示模板
- 所有模板统一展示，无付费标识区分
- 点击模板后右侧预览区立即应用该模板样式

#### 3.4.3 默认模板样式
- 暖色调背景（米黄/奶油色）
- 红棕色标题
- 深绿色列表项圆形编号
- 橙红色bullet点
- 左侧有红色竖线的引用样式
- 页头显示日期数字（大号显示日）+年月+右侧品牌名称
- 页脚显示二维码+品牌名称+描述文字

### 3.5 自定义面板

#### 3.5.1 页面模式
- 整页模式：生成长图
- 分页模式：模拟A4纸张效果
- 通过单选或切换按钮选择

#### 3.5.2 字体大小
- 通过滑块调节正文基准字号
- 实时更新预览区字体大小

#### 3.5.3 行高大小
- 通过滑块调节段落间距
- 实时更新预览区行高

#### 3.5.4 标题字号
- 提供链接/解锁图标切换
- 跟随正文比例：标题字号随正文字号自动调整
- 独立设置：通过滑块单独调节标题字号

#### 3.5.5 表格字号
- 提供链接/解锁图标切换
- 跟随正文比例：表格字号随正文字号自动调整
- 独立设置：通过滑块单独调节表格字号

#### 3.5.6 代码主题
- 提供下拉选择或缩略图选择
- 包含深色、浅色等多种代码配色方案
- 选择后实时更新预览区代码块样式

### 3.6 小组件面板

#### 3.6.1 页头设置
- 开关控制页头显示/隐藏
- 「更换样式」按钮：切换页头布局样式
- 日期设置：
  - 自动获取当前日期
  - 支持手动选择日期
- 自定义文本输入框：输入品牌名称或其他文本（如「妙记文卡」）
- 页头预览样式展示：实时显示页头效果

#### 3.6.2 页脚设置
- 开关控制页脚显示/隐藏
- 「更换样式」按钮：切换页脚布局样式
- 二维码设置：
  - 二维码类型下拉选择
  - 链接地址输入框：输入生成二维码的链接
- 自定义文本1输入框：输入品牌名称或描述文字
- 自定义文本2输入框：输入补充描述文字
- 页脚预览样式展示：实时显示页脚效果

#### 3.6.3 自定义水印管理
- 图片上传入口：点击上传水印图片
- 上传的图片作为水印叠加在卡片上
- 透明度调节：通过滑块调节水印透明度
- 位置选项：选择水印位置（居中、右下角等）

### 3.7 右侧预览区

#### 3.7.1 卡片实时预览
- 实时渲染编辑器内容
- 应用当前选择的模板样式
- 显示自定义的字体、行高、标题字号、表格字号、代码主题
- 显示页头、页脚、水印效果
- 根据页面模式显示整页或分页效果

### 3.8 导出功能

#### 3.8.1 导出按钮
- 位于右上角固定位置
- 点击后弹出导出弹窗

#### 3.8.2 导出弹窗
- 导出范围选择：
  - 当前页
  - 全部页面
- 导出格式选择：
  - PNG
  - JPG
- 清晰度选择：
  - 标准
  - 高清
  - 超清（标注Pro）
- 确认导出按钮：点击后生成并下载图片文件

## 4. 业务规则与逻辑

### 4.1 Markdown解析与渲染
- 编辑器输入内容实时解析为Markdown语法
- 解析结果实时渲染到右侧预览区
- 支持标准Markdown语法：标题（H1-H6）、列表（有序/无序）、引用、加粗、斜体、分割线、图片、表格、代码块
- 支持LaTeX数学公式解析与渲染
- 代码块根据语言类型应用对应语法高亮

### 4.2 样式动态切换
- 用户调整字体大小、行高、标题字号、表格字号、代码主题时，预览区立即更新
- 标题字号和表格字号在「跟随正文」模式下，随正文字号按比例自动调整
- 标题字号和表格字号在「独立设置」模式下，不受正文字号影响

### 4.3 模板应用
- 用户点击模板后，预览区立即应用该模板的背景、颜色、字体、布局等样式
- 模板样式不影响用户自定义的字体大小、行高等设置

### 4.4 页头页脚逻辑
- 页头开关关闭时，预览区不显示页头
- 页头开关开启时，预览区显示页头，包含日期和自定义文本
- 页脚开关关闭时，预览区不显示页脚
- 页脚开关开启时，预览区显示页脚，包含二维码、自定义文本1、自定义文本2
- 二维码根据输入的链接地址生成

### 4.5 水印叠加
- 用户上传水印图片后，图片叠加在卡片内容之上
- 水印透明度根据用户调节的滑块值设置
- 水印位置根据用户选择的位置选项放置

### 4.6 导出逻辑
- 导出范围为「当前页」时，仅导出预览区当前显示的页面
- 导出范围为「全部页面」时，导出预览区所有页面（分页模式下）
- 导出格式为PNG时，生成PNG格式图片文件
- 导出格式为JPG时，生成JPG格式图片文件
- 清晰度为「标准」时，生成标准分辨率图片
- 清晰度为「高清」时，生成高分辨率图片
- 清晰度为「超清」时，生成超高分辨率图片（标注Pro）

## 5. 异常与边界情况

| 场景 | 处理方式 |
|------|----------|
| 编辑器内容为空 | 预览区显示空白卡片或提示「请输入内容」 |
| Markdown语法错误 | 按原文显示，不进行解析渲染 |
| LaTeX公式语法错误 | 显示原始公式文本 |
| 代码块未指定语言 | 使用默认代码高亮样式 |
| 页头日期未设置 | 自动使用当前日期 |
| 页脚链接地址为空 | 不生成二维码或显示占位图 |
| 水印图片上传失败 | 提示上传失败，不显示水印 |
| 导出时网络异常 | 提示导出失败，允许重试 |
| 导出超清清晰度（Pro功能） | 若用户无Pro权限，提示升级或禁用该选项 |

## 6. 验收标准

1. 用户在编辑器中输入Markdown文本
2. 右侧预览区实时显示渲染后的卡片效果
3. 用户在模板面板选择一个模板
4. 预览区立即应用该模板样式
5. 用户在自定义面板调整字体大小和行高
6. 预览区字体和行高实时更新
7. 用户在小组件面板开启页头和页脚，输入自定义文本和链接地址
8. 预览区显示页头和页脚，包含日期、文本和二维码
9. 用户点击右上角「导出」按钮，选择导出范围、格式和清晰度
10. 系统生成并下载对应格式和清晰度的图片文件
