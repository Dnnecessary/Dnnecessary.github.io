## `filePath.trim is not a function` 错误排查
- 用户反馈浏览器一直提示 TypeError: filePath.trim is not a function
- 排查结果：项目源码中没有 filePath 变量和 filePath.trim() 调用
- 该错误来自 Vite dev server 运行时基础设施（rolldown-vite@7.3.1 + miaoda-sc-plugin）
- 原因：miaoda-sc-plugin 的 injected.js 从 HMR 消息中解构 filePath，但 HMR 更新消息中 filePath 可能为 undefined
- 已清除 Vite 缓存（node_modules/.vite），用户需重启 dev server 解决
